# College Football Attendance Analysis

This project analyzes college football attendance trends using SQL in DBeaver and Tableau.

## Tools Used
- SQL
- DBeaver
- Tableau Public

## Files
- `college_football_queries.sql`: SQL queries used for analysis
- `cfb_kpi_summary.csv`: overall KPI summary
- `cfb_attendance_by_season.csv`: average attendance by season
- `cfb_attendance_by_time_bucket.csv`: attendance by game time
- `cfb_ranked_vs_unranked.csv`: ranked vs unranked opponent attendance
- `cfb_top_15_games.csv`: top 15 highest attendance games
- `cfb_top_teams_avg_attendance.csv`: teams with highest average attendance
- dashboard image: final dashboard screenshot

## Key Insights
- Average attendance stays relatively stable over time, with a slight decline in later years.
- Ranked opponents attract more spectators than unranked opponents.
- Day games have the highest average attendance.
- Penn State leads the dataset in average attendance.
- The highest-attended games are dominated by major programs and big matchups.
